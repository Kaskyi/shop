# Shop template
## Installing dependencies
### Main
```
npm install
```
### Frontend
```
cd ./public
npm install
bower install
```

## Start debug
### Prepare to start
## Run mongodb (check mongod location)
```
"C:\Program Files\MongoDB\Server\3.2\bin\mongod.exe"
```
### Run server
```
node server
```
### Run front-end build
```
cd ./public
webpack --watch
```
